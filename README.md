## **Using 📦**

1. Install Packages

```
npm install
```

3. Start Project

```
npm run dev
```

4. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```
rm -rf .git
```

## **Options ✍️**

1. Check lint

```
npm run lint
```

2. Fix lint

```
npm run lint:fix
```

3. Check prettier

```
npm run prettier
```

4. Fix prettier

```
npm run prettier:fix
```

5. Fix lint and prettier

```
npm run format
```
